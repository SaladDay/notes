### 分类算法



#### 1. KNN算法

##### 1.1 介绍

> K近邻（K-Nearest Neighbor, KNN）是一种最经典和最简单的*有监督学习*方法之一。
>
> 定义：如果一个样本在特征空间中的K个最相似（即特征空间中最近邻）的样本中的大多数属于一个类别，那么该样本也属于此类别
>
> 其核心思想：用你的邻居来推断出你的类型

##### 1.2 API

```python
estimator = sklearn.neighbors.KNeighbarsClassifier(n_neighbors=5,algorithm='auto')
```
