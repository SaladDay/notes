

### 无参数装饰器

由于python中没有函数重载，从一个函数名即可唯一分辨一个函数。

```python
def print_start_and_end(func):
  def new_myfunc(*args,**kwargs):
    print("begin")
    func()
    print("end")
  return new_myfunc
-------------------------------------------------------------
@print_start_and_end
def myFunc(a):
  print("myfunc",a)
-------------------------------------------------------------
# 加了装饰器的函数就相当于
def myFunc(a):
  print("myfunc",a)
  
myFunc = print_start_and_end(myfunc)
```
